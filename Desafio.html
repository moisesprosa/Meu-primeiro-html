Entendendo a Comunicação Client x Server. Voltar

Para entender a comunicação entre cliente x server, antes é necessário falarmos sobre o que é cada um desses pontos abordados.

Quando falamos sobre client x server, estamos observando dois lados diferentes de um trabalho que está ocorrendo.

Client: O lado client se trata de todo trabalho/projeto desenvolvido que tem como objetivo a ótica das pessoas, que são considerados clientes do desenvolvedor. Essa visão client se trata de tudo que as pessoas poderão ver através de um navegador em aplicações web.

Server: Por outro lado, o server(servidor) é o que o cliente não está conseguindo ver, onde estão localizados informações e bancos de dados geralmente hospedados na rede da internet.

Introdução ao HTML na Prática Voltar

Nesse conteúdo aprendemos como funciona o desenvolvimento inicial de um documento HTML e também do que se trata essa linguagem de marcação.

É necessário reformar que o HTML não é uma linguagem de progração, mas uma linguagem de marcação como o próprio nome diz.

No HTML é possível fazer diversas edições e também construir páginas robustas para diversos nichos na internet. No entanto é necessário observar alguns pontos e ferramentas básicas para sua criação. Segue alguns exemplos:

Tags:

Uma tag HTML é uma marca que indica ao navegador que um determinado texto ou elemento deve ser interpretado conforme as regras do HTML. As Tags devem ser utilizadas dentro do simbolos de < Menor e > Maior

Exemplos de Tags

p: Indica um parágrafo

H1 ao H6: são recursos de programação HTML utilizados para destacar títulos e subtítulos de uma página. Elas são utilizadas na hierarquização e estrutura de uma página.
